Java图片转换为PDF并合成同一PDF
2019-03-08|java
字数总计: 1.2k|阅读时长: 6 分钟

1.需求

将同一文件夹下的图片和PDF,转换合并为同一个PDF，并要求转换中的图片按格式排版好

2.思路

  1. 调整图片，检测图片大小及方向，调整至A4大小(842×595)， 寻找最佳缩放比，等比例缩放图片，为避免多次IO操作，返回BufferedImage
  2. 将图片转换为PDF
  3. 将文件夹下的PDF按顺序合并，得到最终的final.pdf
